# Pricing — FilePro

Last updated: 2026-06-06

## Free

- Price: $0/month (forever free)
- Signup required: No
- File size limit: 20 MB per file
- Conversions per day: Unlimited (1 file at a time)
- Supported formats: PDF, DOCX, JPG, PNG, MP4 (all formats)
- Compression levels: Low, Medium, High
- File retention: Auto-deleted after 1 hour
- Ads: Yes
- Batch uploads: No
- Priority processing: No
- API access: No

## Premium

- Price: $9.99/month (billed monthly, cancel anytime)
- Signup required: Yes
- File size limit: 500 MB per file
- Conversions: Unlimited
- Supported formats: PDF, DOCX, JPG, PNG, MP4 (all formats)
- Compression levels: Low, Medium, High
- File retention: Auto-deleted after 1 hour
- Ads: No
- Batch uploads: Yes (multiple files at once)
- Priority processing: Yes (faster queue)
- API access: Yes

## Notes

- No long-term contracts. Cancel anytime from account settings.
- Payment processed by Stripe (PCI compliant). Accepts Visa, Mastercard, Amex.
- Access continues until end of billing period after cancellation.
- Free plan requires no email or account — fully anonymous.
- Premium plan unlocks larger file sizes (500MB vs 20MB free limit).
